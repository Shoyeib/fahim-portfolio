IoT Device Based on ESP32

This project is a smart IoT device built with an ESP32 microcontroller. It reads real-time data from sensors and sends it over WiFi for remote monitoring and automation.

🔧 Tools Used

  • ESP32
  • DHT11 Temperature & Humidity Sensor
  • Arduino IDE (C++)
  • Firebase Realtime Database

📸 Screenshots

💡 Key Features

  • Real-time sensor monitoring
  • Cloud connectivity (Firebase)
  • Energy efficient setup
